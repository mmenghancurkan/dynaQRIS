# DynaQRIS (Dynamic QRIS)

## Key Features ⭐
- Generates dynamic QRIS from any merchant
- Set payment status
- Profiles to save static QRIS data
- Show payment history

## Setup 🔧
```python
  git clone https://mmenghancurkan/dynaQRIS.git
  pip install -r requirements.txt
  python app.py
```
